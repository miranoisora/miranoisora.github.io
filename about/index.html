关于我和这个网站

Jun 23, 2017 in About

上次更新：2020-03-31

这个网站是基于 GitHub Pages 和 Hexo 的，所有的照片视频储存在 Amazon S3 OneDrive，其他图片等储存在 GitHub。虽然都有 CDN 分发，但是在朝鲜和其他地区仍可能访问缓慢。此外，会有 YouTube、GoogleMaps、Flickr 之类的链接，上述地区需要代理。

頭像來自 Pixiv / 卯月つくよ

我在台灣逛了幾天之後，接觸了大量的繁體字，也了解很多字型上的差異。其實我在學習少量日語的時候，有一個很頭疼的問題就是那些繁體字很難記住，在從台灣回來之後的這幾個月，我花了不少時間看相關的文章和討論，或是說在學習漢字。我發現沒用太久辨別簡化思想和演變方式之後，之前困擾我的日語漢字好像變得沒有那麼不會寫了，挺容易記住的。我現在基本能知道簡化了哪些字，以及基本能寫所有常用繁體字了。

說一些其中的感觸，一個是我覺得出門去別的地方，很激勵我去學習新的東西，文化地理科學各種都算。另一個是我看到有些人說什麼大陸簡化是沒有章法，日語簡化注重類推，我一開始還覺得大陸簡化好像確實是喔（因為很多草書楷化），比如說對於「檢」，大陸是「检」，日本是「検」，把兩個「口」合併了，有點意思哈，然後其他結構類推比如「験」，還有很多例子「權」->「権」之類的。後來想想這說法簡直屁話，日語簡化漢字什麼結構都可以用三個點代替「櫻」->「桜」，「學」->「学」，「戰」->「戦」。還了解了一點新舊字型，比如「說」和「説」，總之還挺有意思的。

最後說到這個 Blog，之後我應該會使用繁體漢字寫一些內容，反正我用拼音，也沒什麼本質差異。我倒是覺得繁體好看一點，當然，對於筆畫特別多的字，我也會採用合適的簡化字（也就是說包括日本簡化的）替換，不然太小影響閱讀。文字嘛，注重內容，少點「規範」和優越感。另外，我喜歡傳統明體的寫法，所以我會用日語的思源宋體作為首選。

最近进入了一个新的阶段，然后上周末和今天花了三天来彻底把这个网站改成了我之前希望的样子，加上了诸如页内跳转的动画、思源宋体支持、懒加载图片、简化的回到目录按钮等等，还有一些排版上的比如对齐、字符段落间距、字号等等。其实我还是非常关注设计美学的，尽管我的「专业」跟这个毫不相关，当然如何让网页看上去美观也是很重要的事。
遗憾就是，我觉得 2019 的网页应该是会有一些载入动画和加载提示的，以及应该使用 PJAX。但是我目前都还没有好好学过 JavaScript，尝试了挺久，很多东西还是没能力实现，就此作罢，已经花了不少时间。
